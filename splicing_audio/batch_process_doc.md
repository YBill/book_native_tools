# batch_process.py 方案文档

## 功能概述

批量处理 `audio/` 目录下所有 MP3 文件，根据文件名第三段自动选择处理模式，统一输出到 `audio_with_silence/` 目录。

## 处理流程

文件名格式必须严格遵守 `a_b_c.mp3` 规则，脚本取第三段 `c` 判断模式：

```
c == 0：[1秒静音] + [原音频] + [1秒静音] → 输出MP3
c == 1：[1秒静音] + [原音频] + [0.5秒静音] + [effect.mp3] → 输出MP3
```

## 目录结构要求

```
项目根目录/
├── batch_process.py
├── effect.mp3              # 结尾音效文件（必须存在）
├── audio/                  # 源音频目录（放置待处理的 MP3 文件）
│   ├── 10_3_0.mp3          # c=0，前后各加1s静音
│   ├── 10_8_1.mp3          # c=1，开头1s静音+结尾音效
│   └── ...
└── audio_with_silence/     # 输出目录（自动创建）
```

## 配置项

脚本顶部三个路径变量，按需修改：

```python
source_dir  = Path('audio')               # 源音频目录
output_dir  = Path('audio_with_silence')  # 输出目录
effect_file = Path('effect.mp3')          # 结尾音效文件
```

## 文件名规则

| 段位 | 说明 | 示例 |
|------|------|------|
| `a` | 第一段，任意标识 | `10` |
| `b` | 第二段，任意标识 | `3` |
| `c` | 第三段，处理模式，只能为 `0` 或 `1` | `0` |

- 段数不足三段 → `[ERROR]` 跳过
- 第三段不是 `0` 或 `1` → `[ERROR]` 跳过

## 核心技术方案

### 1. 启动前检查

脚本启动时立即检查 `effect.mp3` 是否存在，不存在则直接退出，不进入处理循环。

### 2. 自动检测原音频参数

使用 `ffprobe` 读取每个文件的音频参数，确保静音段与原音频格式匹配：

| 参数 | 说明 | 回退默认值 |
|------|------|-----------|
| `sample_rate` | 采样率（如 44100、16000） | 44100 |
| `channels` | 声道数（1=单声道，2=立体声） | 2 |
| `bit_rate` | 码率（优先取 stream，其次取 format） | 192k |

### 3. FFmpeg 拼接命令

**mode=0** 输入流索引：

| 索引 | 内容 | 时长 |
|------|------|------|
| `[0:a]` | 原音频 | 原始时长 |
| `[1:a]` | 开头静音（anullsrc） | 1 秒 |
| `[2:a]` | 结尾静音（anullsrc） | 1 秒 |

```
[1:a][0:a][2:a]concat=n=3:v=0:a=1[out]
```

**mode=1** 输入流索引：

| 索引 | 内容 | 时长 |
|------|------|------|
| `[0:a]` | 原音频 | 原始时长 |
| `[1:a]` | 开头静音（anullsrc） | 1 秒 |
| `[2:a]` | 结尾静音（anullsrc） | 0.5 秒 |
| `[3:a]` | effect.mp3 | 音效原始时长 |

```
[1:a][0:a][2:a][3:a]concat=n=4:v=0:a=1[out]
```

### 4. 输出编码

- 编码器：`libmp3lame`
- 码率：与原音频一致（自动检测）
- 采样率：与原音频一致（anullsrc 参数匹配）
- 声道数：与原音频一致（anullsrc channel_layout 匹配）

## 输出结果说明

| 项目 | 说明 |
|------|------|
| 输出文件名 | 与原文件名相同 |
| 输出位置 | `audio_with_silence/` 目录 |
| 时长变化（mode=0） | 原时长 + 2 秒（开头1秒 + 结尾1秒） |
| 时长变化（mode=1） | 原时长 + 1.5秒 + effect.mp3 时长 |
| 格式 | 与原音频完全一致（编码/采样率/声道/码率） |

## 依赖环境

- Python 3.x
- FFmpeg（需在系统 PATH 中，包含 `ffmpeg` 和 `ffprobe` 命令）

## 错误处理

| 情况 | 处理方式 |
|------|---------|
| `effect.mp3` 不存在 | 启动时立即退出，不处理任何文件 |
| `audio/` 目录为空 | 启动时立即退出 |
| 文件名段数不足3段 | `[ERROR]` 跳过，计入失败数，继续下一个 |
| 第三段不是 `0` 或 `1` | `[ERROR]` 跳过，计入失败数，继续下一个 |
| ffprobe 获取参数失败 | 使用默认值（44100Hz, 立体声, 192kbps）并继续处理 |
| ffmpeg 处理失败 | 输出错误信息（stderr 前200字符），计入失败数，继续下一个 |
